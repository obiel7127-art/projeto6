Este projeto é uma calculadora funcional desenvolvida com HTML, CSS e JavaScript, incluindo operações básicas e recursos úteis.
Ela possui:

Display dinâmico para mostrar números e resultados

Botões numéricos de 0 a 9

Operações: +, -, *, /

Funções extras:

CE – limpa apenas o display

C – limpa todo o cálculo

Backspace – apaga o último caractere

± – inverte o sinal

, – adiciona decimal

= – calcula o resultado

A lógica da calculadora é controlada no arquivo calculadora.js, enquanto o estilo visual está em style.css.
